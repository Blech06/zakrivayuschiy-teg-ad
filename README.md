# Закрывающий тег

Персональный проект-рефлексия по итогам учебного курса.  
В карточках собраны мои мысли, переживания и яркие моменты обучения вёрстке и анимации.

## Опубликованная версия

[Открыть сайт](https://твой-username.github.io/название-репозитория/index.html)

## Как запустить локально

1. Склонируй репозиторий
2. Открой `index.html` в браузере

## Технологии

- HTML5, CSS3, вариативные шрифты
- CSS-анимации (сердце, кнопки)
- CSS-фильтры и режимы смешивания
- Модальное окно на `<dialog>`
- Адаптивная вёрстка

---

### О проекте
Этот сайт — цифровой дневник. Каждая карточка рассказывает об одном этапе пути: от первых шагов до сложных задач, с которыми удалось справиться. Фотографии подобраны под настроение, а фильтры усиливают нужную эмоцию.
